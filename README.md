# catsVSdogs
Clasificación de imágenes "Perro/Gato".
Clasificación de Imágenes de Perros y Gatos utilizando un modelo pre-entrenado de Keras. Su función es: cargar un modelo de Machine Learning, usarlo para predecir la clase (perro o gato) de imágenes individuales y en lotes, y finalmente evaluar la precisión del modelo en un conjunto de imágenes de prueba.
